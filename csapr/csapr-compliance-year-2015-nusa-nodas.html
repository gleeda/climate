February 12, 2016 - Round 2 Final NODA for 2015 Allowance Allocations: New Units for All CSAPR Programs Except NOX Ozone Season.

This NODA details compliance year 2015 allowance allocations to certain new units eligible for a "2nd Round" CSAPR Nitrogen Oxide (NOx) Annual, Group 1 Sulfur Dioxide (SO2), or Group 2 SO2 new unit set-aside allocation. It also details 2015 allowance allocations to CSAPR existing units in states in which the new unit set-asides for 2015 for those annual CSAPR Trading Programs were undersubscribed.

  • Notice of Data Availability Federal Register (81 FR 7466 February 12, 2016)
  • New Unit Set Aside (NUSA) 2015 Nitrogen Oxides (NOx) Annual 2nd Round(23 pp, 112 K, February 4, 2016) (Excel)
  • Allocations to Existing Units of Remaining New Unit Set Aside Allowances from 2015 CSAPR NOx Annual Trading Program(23 pp, 249 K, February 4, 2016) (Excel)
  • New Unit Set Aside (NUSA) 2015 Sulfur Dioxide (SO2) Group 1 and SO2 Group 2 2nd Round (23 pp, 62 K, February 4, 2016) (Excel)
  • Allocations to Existing Units of Remaining New Unit Set Aside Allowances from 2015 CSAPR SO2 Group 1 and SO2 Group 2 Trading Programs(23 pp, 246 K, February 4, 2016) (Excel)

November 12, 2015 - Round 2 Final NODA for 2015 Allowance Allocations:  New Units for CSAPR NOX Ozone Season

This NODA details 2015 allowance allocations to certain new units eligible for a “2nd round” CSAPR ozone season new unit set-aside allocation. It also details allowance allocations to CSAPR ozone season existing units in states in which the new unit set-asides for the 2015 CSAPR ozone season were undersubscribed.

  • Notice of Data Availability Federal Register (80 FR 69883 November 12, 2015)
  • New Unit Set Aside (NUSA) 2015 Nitrogen Oxides (NOx) Ozone Season 2nd Round(25 pp, 64 K, November 9, 2015) (Excel)
  • Allocations to Existing Units of Remaining New Unit Set Aside (NUSA) Allowances from 2015 CSAPR ozone season(25 pp, 279 K, November 9, 2015) (Excel)
