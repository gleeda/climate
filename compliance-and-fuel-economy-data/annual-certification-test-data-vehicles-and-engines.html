Annual Certification Test Data for Vehicles and Engines

Overview

The Annual Certification Test Results Report (often referred to as Federal Register Test Results Report) includes light-duty vehicle and heavy-duty engine reports for model years 1979 through 1994 and light-duty only data for later model years.  Heavy-duty engine data for 1995 and 1996 is also available. The report contains certification levels (projected emission levels at the end of the useful life miles of a vehicle) and deterioration factors used to compute the certification levels.

The formatted report is not available for 1999 and later model years. The list of Certified Vehicle Models can be used in conjunction with the Certified Vehicle Test Result Report Data file by matching the Engine Family/Test Group and Evaporative Family combinations for a given vehicle model with the test results for that same combination in each file. The vehicle model name appearing in the Certified Vehicle Test Result Report Data does not necessarily represent the actual vehicle that was tested.
